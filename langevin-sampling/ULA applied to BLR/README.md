# Unadjusted Langevin Algorithm (ULA) applied to Bayesian Logistic Regression

This project contains Jupyter notebooks illustrating the application of the Unadjusted Langevin Algorithm (ULA) to Bayesian logistic regression (BLR).

The notebooks focus on the implementation and numerical evaluation of ULA for posterior sampling and parameter estimation using real-world datasets. They provide reproducible examples of Bayesian inference based on Langevin dynamics.

## Contents

- Implementation of the Unadjusted Langevin Algorithm (ULA)
- Parameter estimation
- Two numerical experiments and visualization of results
