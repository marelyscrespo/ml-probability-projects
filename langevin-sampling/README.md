# Langevin Sampling Algorithms

Implementation of stochastic sampling methods for Bayesian inference.

## Methods
- Unadjusted Langevin Algorithm (ULA) / Langevin Monte Carlo (LMC)
- Stochastic Gradient Langevin Dynamics (SGLD)

## Content
- Simulation of stochastic differential equations
- Numerical experiments on convergence
- Scalability analysis in high dimension

## Tools
Python, NumPy, SciPy, Matplotlib
